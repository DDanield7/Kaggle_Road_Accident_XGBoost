# Kaggle_Road_Accident_XGBoost
Kaggle competition predictions on a road accident dataset, utilizing XGBoost with Stratified K-Fold CV and Optuna for hyperparameter tuning.

# Workflow
#### 1. Data Preparation
  - 12 features of environmental factors
  - 1 target feature of accident risk
#### 2. Model Training
  - using optuna for hyperparameter tuning
#### 3. Model Evaluation
  - Metric used: RMSE
  - Stratified K-Fold CV K=7

# Result
Mean RMSE : 0.0566
